<span id="firstline">g. Write a script to open window having specific width, height and with a status bar on button click.</span><br><br>

**Code:**

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <button id="window_open" onclick="openWindow()">Open</button>
    <script>
        function openWindow() {
            open("https://www.google.com", width="500", height="200")
        }
    </script>
</body>
</html>
```
